# Hangman Game (Python)

A simple text-based Hangman game built using Python.
The player guesses a hidden word one letter at a time with a limited number of attempts.

## Features
- Uses a predefined list of words
- Random word selection
- Maximum of 6 incorrect guesses
- Console-based interaction
- Shows hints by revealing guessed letters
- No word reveal on game over

## Technologies Used
- Python 3
- random module
- Loops and conditional statements
- Lists and strings

## How to Run
1. Make sure Python 3 is installed
2. Clone this repository or download the code
3. Run the file using:

python hangman.py

## Game Rules
- Guess one letter at a time
- Repeated or invalid inputs are ignored
- The game ends when:
  - All letters are guessed correctly (Win)
  - All attempts are used (Game Over)
